# Інструкція: Як підключити додаток до n8n

Щоб ваш мобільний додаток міг спілкуватися з n8n, вам потрібно створити простий Workflow (сценарій) з 2 або 3 вузлів (nodes).

## Крок 1: Створіть вузол Webhook (Тригер)
1. Додайте вузол **Webhook**.
2. В налаштуваннях вузла вкажіть **HTTP Method**: `POST`.
3. Залиште **Path** за замовчуванням (або впишіть щось на кшталт `chat`).
4. **Respond**: Змініть на `Using 'Respond to Webhook' Node` (це дуже важливо, щоб ми могли сформувати власну відповідь).
5. Скопіюйте **Test URL** (або Production URL) — це саме те посилання, яке потрібно вставити у ваш файл `app/(tabs)/chat.tsx` (змінна `N8N_WEBHOOK_URL`).

## Крок 2: Обробка повідомлення (Вузол Code)
Мобільний додаток відправляє повідомлення у форматі JSON:
```json
{
  "message": "Привіт, як справи?"
}
```

1. Додайте вузол **Code** одразу після Webhook.
2. Скопіюйте код з файлу `n8n_code_script.js` (який я щойно створив у вашій папці) і вставте його у цей вузол.
3. Цей код прочитає повідомлення і створить об'єкт для відповіді.

*(Опціонально: Замість вузла "Code" ви можете використати вузли OpenAI/Anthropic/Basic LLM Chain, щоб отримувати справжні відповіді від штучного інтелекту. Тоді вам треба буде передати їм текст `$json.body.message` і отримати результат).*

## Крок 3: Повернення відповіді (Вузол Respond to Webhook)
Додаток очікує отримати назад JSON з полем `reply`.

1. Додайте вузол **Respond to Webhook**.
2. У полі **Respond With** виберіть `JSON`.
3. У полі **Response Body** вставте вихідні дані з попереднього вузла:
   `{{ $json }}`
   (Або якщо ви використовуєте AI вузол, сформуйте JSON руками: `{ "reply": "{{ $json.output }}" }`).

## Як протестувати?
1. Вставте ваш Webhook URL у файл `chat.tsx`.
2. В n8n натисніть **Listen for Test Event**.
3. Відкрийте ваш додаток на телефоні/симуляторі і відправте будь-яке повідомлення в чат.
4. n8n зловить запит, пропустить через код і миттєво відправить відповідь назад у ваш мобільний додаток!
